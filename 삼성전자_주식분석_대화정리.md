## 4. PER vs PBR 개념 비교

### Q. PBR, PER 개념 비교해줘

#### 공식 정리

| 지표      | 공식              | 기준        |
| ------- | --------------- | --------- |
| **PER** | 주가 ÷ EPS(주당순이익) | **이익** 기준 |
| **PBR** | 주가 ÷ BPS(주당순자산) | **자산** 기준 |

#### 직관적 이해

| 지표        | 의미                          |
| --------- | --------------------------- |
| **PER**   | "이 회사가 1년에 버는 돈 대비 얼마나 비싸냐" |
| PER 10배   | 지금 이 속도로 벌면 10년 후 본전        |
| PER 37배   | 시장이 "앞으로 훨씬 더 벌 것"이라고 베팅    |
| **PBR**   | "이 회사의 장부상 재산 대비 얼마나 비싸냐"   |
| PBR 1배    | 지금 회사 청산하면 주가와 딱 맞음         |
| PBR 1배 미만 | 장부가보다 싸게 거래 → 이론상 저평가       |

#### 언제 어떤 지표를 쓰나

| 업종          | 주로 쓰는 지표        | 이유               |
| ----------- | --------------- | ---------------- |
| 성장주 (AI·전력) | **PER**         | 자산보다 이익 성장이 핵심   |
| 반도체·조선·건설   | **PBR**         | 사이클 산업, 순이익 들쭉날쭉 |
| 금융·보험       | **PBR**         | 자산 규모가 본질 가치     |
| 적자 스타트업     | **PSR** (매출 기준) | 이익 자체가 없음        |

#### 한 줄 요약

> **PER = 수익성 온도계** / **PBR = 자산 안전마진 온도계**

## 6. EPS 개념 정리

### Q. EPS가 뭐지?

**EPS (Earnings Per Share) = 주당순이익**

**공식:** `EPS = 당기순이익 ÷ 발행주식수`

> 삼성전자 2026F: 순이익 135조 ÷ 발행주식 60억주 = **EPS 22,316원**
> → 주식 1주당 1년에 2만 2천원을 번다는 의미

#### PER과의 관계

```
PER = 주가 ÷ EPS

161,000원 ÷ 22,316원 = PER 7.2배
```

- **EPS 오르면 → PER 내려감** (주가 그대로여도)
- 삼성전자 PER 7배대 이유: EPS가 2023년 2,424원 → 2026F 22,316원으로 **9배 폭등**했는데 주가는 그만큼 안 올랐기 때문

#### EPS 해석

| 상황     | 의미                     |
| ------ | ---------------------- |
| EPS 높다 | 1주당 많이 번다 → 주주 입장에서 좋음 |
| EPS 낮다 | 수익성 약함 or 불황           |
| EPS 급증 | PER 자동 하락 → 저평가처럼 보임   |

> **EPS는 PER의 분모** — 이걸 알면 PER 움직임이 바로 읽힌다

## 7. 4개 지표 정규화 교차 선 차트 (시각화 #2)

### Q. 4개를 선 차트로 한번에 넣어줘 — 선의 교차점을 보려는 거야

#### 정규화(Normalization) 적용 이유

4개 지표(PER·PBR·EPS·주가)는 단위와 스케일이 완전히 달라 동일 축 비교 불가.
→ 각 지표를 **0~100 범위로 정규화**해 방향성과 교차점만 시각화.

```
정규화 공식: (현재값 - 최솟값) / (최댓값 - 최솟값) × 100
```

#### 정규화 기준값

| 지표  | 최솟값     | 최댓값      |
| --- | ------- | -------- |
| PER | 7.2배    | 32.4배    |
| PBR | 0.9배    | 2.6배     |
| EPS | 2,424원  | 22,316원  |
| 주가  | 52,900원 | 161,000원 |

#### 핵심 교차 포인트 분석

| 구간             | 교차 내용            | 투자 시사점                                           |
| -------------- | ---------------- | ------------------------------------------------ |
| **2023→2024**  | PER 급락 (32→9배)   | 반도체 불황 탈출, 이익 회복. PBR 1배 이하 저점                   |
| **2024→2025**  | 주가·PBR 역전 급등     | 주가 3배 폭등(5.3만→16만), EPS보다 주가 선행                  |
| **2025→2026F** | **EPS선이 주가선 추월** | EPS +199% 폭발, 주가 제자리 → PER 21→7배 급락 = **저평가 신호** |

#### 결론

> 2025→2026F 구간에서 EPS 정규화 선이 주가 정규화 선을 추월하는 교차 발생
> → "이익이 주가를 앞질렀다" = **이익 성장이 주가에 아직 미반영 = 저평가 가능성**

## 📌 전체 대화 핵심 요약

### 오늘 다룬 개념 정리

| 개념      | 공식        | 핵심                       |
| ------- | --------- | ------------------------ |
| **PER** | 주가 ÷ EPS  | 수익성 온도계. 낮을수록 이익 대비 싸다   |
| **PBR** | 주가 ÷ BPS  | 자산 안전마진. 1배 이하 = 이론적 저평가 |
| **EPS** | 순이익 ÷ 주식수 | PER의 분모. 급등하면 PER 자동 하락  |

### 종목별 현재 밸류에이션 한줄 정리

| 종목           | PER   | 성격                        |
| ------------ | ----- | ------------------------- |
| **HD현대일렉트릭** | ~37배  | 성장주 프리미엄, 전력 인프라 슈퍼사이클    |
| **삼성전자**     | ~7.2배 | 이익 폭발 대비 주가 미반영, 이익 성장 추종 |
| **SK하이닉스**   | ~6.8배 | HBM 독점 지위, PER 역전 이슈      |

### 투자 관점 시사점

- 반도체 2종목은 **PBR 기준**으로 봐야 본질 가치 파악 가능
- 삼성전자 EPS가 주가를 앞지른 구간 → 이익 성장 선반영 여부가 핵심 변수
- HD현대일렉트릭은 PER 37배지만 **이익 성장률이 정당화**하는 구조

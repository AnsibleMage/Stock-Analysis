## 1. ADR 상장 구조 이해

### ARM의 ADR 상장 케이스

- ARM은 **ADR(American Depositary Receipt)** 방식으로 NASDAQ에 상장
- 원주는 런던거래소(LSE)에 상장되어 있지 않음 — 2016년 소프트뱅크 인수 후 LSE 상장폐지
- 2023년 9월 14일, 비상장 상태에서 NASDAQ에 ADR로 재상장 → **차트는 이날부터 시작, 이전 기록 없음**
- 즉 "본국 거래소 연속 차트가 없고, IPO일이 차트 Day 1"인 신규 IPO형 ADR 케이스

### 케이스 구분

| 케이스             | 설명                                       |
| --------------- | ---------------------------------------- |
| 신규 ADR 생성       | 본국 주식 기반으로 새로 ADR 발행 → 미국 차트는 상장일부터 시작   |
| 기존 거래소 이전/업그레이드 | OTC에서 NASDAQ 이전 시 → OTC 시절 데이터 이어질 수도 있음 |

## 2. SK하이닉스 ADR 상장

### 상장 일정 및 규모

- **상장 예정일: 2026년 7월 10일** (신주 상장 7/29, 청약·납입일 7/14)
- 최대 1,779만 주 신규 발행(전체 주식 약 2.5%), 최대 45조 4,534억 원 조달
- 주관사: BofA, 씨티, 골드만삭스, JP모건

### ADR 구조

- **ADR 1단위 = 원주 0.1주** (원주 1주를 10등분)
- 이유: 원주 1주 달러 환산 시 약 $1,740으로 고가 → 투자자 접근성 확보

### 자금 용도

- 용인 반도체 클러스터 1단계 팹(Y1), 청주 P&T7 첨단 패키징 팹, EUV 장비 투자

### 기대 효과

- 필라델피아 반도체지수(SOX) 편입 가능성 → 패시브 자금 유입 기대
- 마이크론 선행 PER 11.2배 vs SK하이닉스 6.6배 → 밸류에이션 갭 재평가 기대

### 개인투자자 매수 방법

- 국내 증권사(키움 등) 해외주식 계좌로 7/10 나스닥 개장(한국시간 밤 10:30~) 후 매수 가능
- 이번 공모는 해외 기관 대상 북빌딩 → **일반 개인 공모 청약 없음**
- 키움 영웅문S 글로벌: 해외주식 → 예약주문에서 시장가/지정가 예약 설정 가능
- **단, 상장 첫날 시장가 예약매수는 리스크 큼** — 호가 스프레드가 크고 변동성 심함 → 지정가 권장

## 8. 병목 사이클 흐름 검증

### 맞는 부분

- 1막(LLM)→2막(GPU+HBM)→3막(전력) 순서는 역사적으로 실제 검증됨
- 5막 광통신 병목도 현실로 확인 중 (엔비디아 $40억 투자가 증거)

### 흠 잡히는 부분

1. **4막(로봇)이 5막(광통신)보다 먼저라는 순서가 애매함** — 실제로는 광통신이 이미 터지는 중이고 로봇은 초기 단계
2. **6막 "반도체 2라운드"는 아직 추론 영역** — 2028~2030은 너무 멀어 지금 종목 특정 무리
3. **막 사이 경계가 실제론 겹침** — 순차가 아니라 2·3·5막이 동시 진행 중

### 수정된 그림

```
확인된 현실       지금 동시 진행        추론 영역
LLM(완료)   →   GPU+HBM  ─┐
                전력인프라  ├─ 동시 진행 중
                광통신      ─┘
                              ↓
                         로봇 (초기 진입)
                              ↓
                    엣지 반도체 (2028~, 추론)
```

**결론:** 큰 방향은 맞으나 "순차 막"이 아니라 "동시 레이어"이며, 4·6막은 아직 추론 영역.

## 15. ARM 고객사 독립 리스크

### 문제 제기

"삼성·테슬라·로봇가전·자동차가 ARM에서 독립하면 여긴 망하는 거 아니냐 — 체리사 라이센스 종료 사례처럼"

### 삼성전자 — 이미 ARM 끊기 시작(2026.4)

- 차세대 기업용 SSD 컨트롤러 'BM9K1'에 **RISC-V(오픈소스, 로열티 무료) 전면 적용** — 시제품 아닌 **양산 제품 최초 도입**
- 이유: ARM 로열티는 출하량 비례 부과 → 수천만 대 팔리는 SSD 시장에서 누적 비용이 수익성 직접 압박
- 업계 관측: SSD → 서버 SSD → 궁극적으로 엑시노스(모바일 AP)까지 확대 가능성 주목

### 테슬라 — 이미 부분적으로 ARM 밖

- FSD칩 CPU는 ARM Cortex-A72 기반이나, RISC-V를 ARM IP와 혼용하는 추세
- 완성차 업계 전체가 RISC-V로 이동(로열티 부담이 큰 차량용 반도체 특성)

### 애플 사례 — 이미 증명된 선례

- Apple Silicon은 명령어셋(ISA)만 ARM, 마이크로아키텍처는 애플 완전 독자 개발
- ARM Holdings 스스로도 "Apple과 ARM칩은 서로 크게 상관없다"고 인정

### ARM의 방어 전략

1. 아키텍처 라이센스는 협력사의 자체 커스텀까지 허용 — 완전 이탈이 생각보다 어려움
2. **ARM이 직접 칩 제조까지 진출 선언**(2026.3, CEO 르네 하스) — "IP 라이센스 넘어 완성 칩까지 판매"
3. 엔비디아는 이미 ARM 지분 전량 매각(2026년 초) — 독점력 약화에 대한 시장 베팅 신호
4. RISC-V의 한계 — 아직 범용 AP급 성능 미달, 파편화 문제(비준 확장 40개 이상)

### 정직한 결론

- 삼성의 SSD 이탈은 **패턴의 시작**으로 봐야 함
- 단, 모바일 AP(엑시노스)는 OS 호환성 문제로 아직 이탈 안 함
- ARM의 극단적 고평가(PER 100배)에 대한 애널 다운사이드 컨센서스는 이 독립 리스크가 이미 일부 반영된 결과일 가능성
